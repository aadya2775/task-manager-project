# MERN Stack Task Management System

A robust, full-featured task management platform built with the MERN stack (MongoDB, Express.js, React, Node.js). This project enables secure authentication, advanced task management, real-time team collaboration, and a responsive user interface for both admins and users.

---

## 🚀 Features

- **User Authentication & Authorization**  
	Secure login system with role-based access (Admin/User).

- **User Dashboard**  
	Personalized dashboard displaying assigned tasks, progress tracking, and actionable insights.

- **Advanced Task Management**  
	Create, read, update, and delete (CRUD) tasks with support for due dates, priorities, and detailed descriptions.

- **Automated Status Updates**  
	Task status updates automatically based on checklist completion.

- **Team Collaboration**  
	Assign tasks to multiple team members and track their completion in real-time.

- **Priority & Progress Tracking**  
	Categorize tasks by priority (Low, Medium, High) and monitor progress with visual indicators.

- **Task Report Downloads**  
	Export task data for offline analysis and reporting.

- **Attachments Support**  
	Add and access task-related file links (Google Drive, Figma, PDFs, etc.).

- **Fully Mobile Responsive UI**  
	Seamless experience across desktop, tablet, and mobile devices.

- **Admin Panel**  
	Admins can manage all users and tasks across the platform.

---

## 🛠️ Tech Stack

- **Frontend:** React, Redux Toolkit, Redux Persist, Axios, Vite
- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Authentication:** JWT, Role-based access
- **Visualization:** Recharts (Pie & Bar Charts)
- **File Uploads:** Multer

## 📦 Getting Started

1. **Clone the repository**
2. **Install dependencies**  
	 - `cd backend && npm install`  
	 - `cd ../frontend && npm install`
3. **Configure environment variables**  
	 - Create `.env` files in both `backend` and `frontend` as per sample or documentation.
4. **Run the backend server**  
	 - `cd backend && npm start`
5. **Run the frontend app**  
	 - `cd frontend && npm run dev`
6. **Access the app**  
	 - Visit `http://localhost:3000` (or as configured)
